# LiveSearch React

## Instructions

### Download repository

```bash
git clone https://github.com/webbegg/livesearch-react
```

### Install dependencies

```bash
cd livesearch-react
yarn # or npm install
```

### Run in dev mode

```bash
yarn start # or npm start
```

## Run App with example

```bash
cd livesearch-react
yarn # or npm install
yarn build # or npm run build
cd example
yarn start # or npm start
```
