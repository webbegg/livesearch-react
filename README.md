# ui-search-component

## Install

```bash
npm install --save ui-search-component
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'ui-search-component'
import 'ui-search-component/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
